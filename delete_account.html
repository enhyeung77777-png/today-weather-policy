데이터 삭제 요청

오늘날씨 (Simple Weather)

오늘날씨는 회원가입이 없는 앱으로, 서버에 개인정보를 저장하지 않습니다. 모든 데이터는 기기 내에만 저장됩니다.

기기 내 데이터 직접 삭제

앱에 저장된 모든 로컬 데이터를 삭제하려면 아래 방법을 이용하세요.

1
기기의 설정 앱을 엽니다.
2
앱 (또는 애플리케이션 관리) 메뉴로 이동합니다.
3
오늘날씨를 찾아 선택합니다.
4
저장공간 → 데이터 삭제를 탭합니다.

또는 앱을 기기에서 삭제(언인스톨)하면 모든 로컬 데이터가 자동으로 제거됩니다.

삭제되는 데이터 항목

  • 사용자 설정값 (온도 단위, 테마 등)
  • 캐시된 날씨 데이터
  • 저장된 위치 정보

참고: 삭제된 데이터는 복구할 수 없습니다. 앱을 재설치하면 초기 상태로 시작됩니다.
